## iOS Weather AQI US Standard

### How to use?

1️⃣ Add this to `[rewrite_remote]`
```conf
https://raw.githubusercontent.com/haukeng/proxy-rule/main/quanx/script/aqi-us.conf, tag=😷AQI, update-interval=86400, enabled=true
```

2️⃣ Register an account at https://aqicn.org/data-platform/token/ and open the link from comfirm email.

3️⃣ Everything is ok if you see a notification show `✅ Get token successfully!`.
